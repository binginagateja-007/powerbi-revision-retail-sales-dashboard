# 📊 Retail Sales Performance Dashboard (Power BI Revision Project)

## 📌 Overview

This project was built as a Power BI revision exercise after completing my academic learning. The objective was to strengthen my understanding of Power Query, Data Modeling, DAX, and Dashboard Development by creating an interactive business dashboard from scratch without relying on tutorials.

---

## 🎯 Objective

Develop a retail sales dashboard that provides business insights into:

- Revenue Performance
- Profitability
- Order Analysis
- Regional Performance
- Product Category Performance
- Customer Ratings

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Excel

---

## 📂 Dataset

A retail sales dataset containing:

- Order Details
- Product Categories
- Regions
- Revenue
- Cost
- Quantity
- Ratings
- Order Status
